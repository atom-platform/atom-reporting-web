Donut Gauge Configuration


    <ngx-donut-gauge [value]="value" [config]="config"></ngx-donut-gauge>
value A number between 0 and 1 inclusive that will be displayed as a percentage in the chart.
config Customization configuration for the chart. See example further down.

Sample Custom Configuration


    // all fields are optional to override
    // the gauge uses defaults if the config field is not overridden
    config = {
      strokeWidth: DonutWidth.Thick,
      gradientDirection: DonutGradientOrigin.BottomLeft,
      gradientColors: ['#9d11af', '#309aad', '#cef995'],
      baseColor: '#dae5c9',
      textColor: '#309aad',
      label: 'testLabel',
      title: 'title'
    };

To get constants that strokeWidth and gradientDirection use, please import them from donut-gauge.constants.ts.

Custom Parameters:
strokeWidth Width of the stroke. See donut-gauge.constants.ts for possible values.
gradientDirection The starting direction of the linear gradient for the colors used in the gauge. See donut-gauge.constants.ts for possible values.
gradientColors Takes array of colors in hexadecimal strings prefixed by with #. The first color in the array is the color at the source of the gradient and the rest goes in order. For a solid color, supply only 1 color in the array.
baseColor The color of the gauge base. Is grey by default. Takes a hexadecimal string prefixed with #.
textColor The color of the percentage value and label in the middle. Is black by default. Takes a hexadecimal string prefixed with #.
label Optional text label under the percentage.
title SVG Chart Title for accessibility purposes.
